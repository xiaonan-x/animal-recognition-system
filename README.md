# 动物产生式识别系统（Animal Production Recognition System）

一个基于产生式规则的动物识别系统，通过图形用户界面（GUI）实现特征选择与推理过程可视化，可识别虎、金钱豹、斑马等7种常见动物。

## 🌟 项目介绍

本项目是一个经典的人工智能产生式系统实现，核心功能是根据用户输入的动物特征（如"有毛发"、"会飞"等），通过预设的15条产生式规则进行自动推理，最终识别出动物种类，并展示完整的推理过程。

系统采用Python的Tkinter库构建交互界面，操作简单直观，适合作为人工智能基础课程的实验案例，帮助理解产生式系统的规则库、综合数据库和推理机工作原理。

## 🚀 功能特点

- **可视化交互**：通过复选框选择动物特征，按钮触发推理，结果实时展示
- **完整推理过程**：记录并显示规则匹配顺序、中间结论推导过程
- **7种动物识别**：支持识别虎、金钱豹、斑马、长颈鹿、鸵鸟、企鹅、信天翁
- **即时反馈**：未选择特征时提示用户，特征不足时显示推理失败原因
- **一键重置**：快速清空选择内容，重新开始特征输入

## 📋 支持的特征与规则

### 特征列表
- 基础特征：有毛发、有奶、有羽毛、会飞、会下蛋、吃肉、有犬齿、有爪、眼盯前方、有蹄、反刍动物
- 形态特征：黄褐色、身上有暗斑点、身上有黑色条纹、长脖子、长腿、有黑白二色
- 行为特征：不会飞、会游泳、善飞

### 核心规则（部分示例）
- r1: 有毛发 → 哺乳动物
- r3: 有羽毛 → 鸟
- r5: 吃肉 → 食肉动物
- r9: 哺乳动物 + 食肉动物 + 黄褐色 + 身上有暗斑点 → 金钱豹
- r13: 鸟 + 长脖子 + 长腿 + 不会飞 + 有黑白二色 → 鸵鸟

完整规则库包含15条产生式规则，严格对应经典动物识别问题逻辑。

## 🛠️ 安装与使用

### 环境要求
- Python 3.x（无需额外安装依赖库，Tkinter为Python标准库）

### 快速启动
1. 克隆仓库到本地
   ```bash
   git clone https://github.com/yourusername/animal-recognition-system.git
   cd animal-recognition-system
   ```

2. 直接运行主程序
   ```bash
   python animal_recognition.py
   ```

3. 操作步骤
   - 在左侧勾选动物特征（可多选）
   - 点击"开始推理"按钮，右侧将显示推理过程和结果
   - 如需重新选择，点击"重置选择"按钮清空所有内容

## 🔍 推理示例

**识别长颈鹿的过程：**
1. 选择特征：有奶、有蹄、长脖子、长腿、身上有暗斑点
2. 推理步骤：
   - 匹配规则r2（有奶 → 哺乳动物）
   - 匹配规则r7（哺乳动物 + 有蹄 → 有蹄类动物）
   - 匹配规则r11（有蹄类动物 + 长脖子 + 长腿 + 身上有暗斑点 → 长颈鹿）
3. 输出结果：推理成功，识别为"长颈鹿"
